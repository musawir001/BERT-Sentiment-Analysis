# BERT Sentiment Analysis

Three-class sentiment analysis (Positive, Neutral, Negative) using BERT with confidence score output.

## Features

- BERT-based sentiment classification
- Neutral detection using probability difference
- Confidence score output
- Interactive Gradio interface
- Runs locally without internet
